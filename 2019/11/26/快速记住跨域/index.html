快速记住跨域

Posted on 2019-11-26

在django的settings配置文件中我们常常需要进行跨域的配置来实现 前后端交互，而我们一般都记不住跨域需要配置的那几行代码，这时候就需要我来给你阐述一下怎么记？

首先在 INSTALLED_APPS 中配置的corsheaders,这个我们肯定可以记住，主要就是下面几步

MIDDLEWARE中间件第三行的我们可以这样记

  • from corsheaders.middleware import CorsMiddleware

我们可以这样进行导包，然后将from和import 去掉，把import换成 . 即可

CORS_ORIGIN_ALLOW_ALL = True

这个我们可以在底层代码中找到直接复制粘贴，或者直接改底层代码，改为True即可

  • from corsheaders.middleware import CorsMiddleware
  • 按住ctrl键然后点击CorsMiddleware进入底层
  • 进入后从最上面的导包那里找到 from corsheaders.conf import conf
  • 然后再次按住ctrl键点击conf进入底层的底层
  • 进入后在第30行我们就可以找到CORS_ORIGIN_ALLOW_ALL
  • 最后怎么做取决于自己，可以直接改为true，以后都不需要写这一步，也可以进行复制粘贴到settings配置中
